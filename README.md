¡Bienvenido/a a mi repositorio de aprendizaje! Este espacio contiene todos los ejercicios, prácticas de laboratorio y notas de código desarrolladas durante mi formación en el **Samsung Innovation Campus**, impartido en colaboración con la **Universidad Politécnica de Valencia (UPV)**.

El objetivo de este repositorio es documentar mi evolución técnica, mantener un histórico de buenas prácticas y servir como base de conocimiento para futuros proyectos.

## 📂 Estructura del Repositorio

El contenido está organizado por módulos temporales y temáticos para facilitar la navegación:

* 📁 `modulo-01-fundamentos/` → Introducción a la sintaxis, estructuras de datos y lógica de programación.
* 📁 `modulo-02-datos/` → Manipulación de datos, consultas en bases de datos y limpieza de datasets.
* 📁 `modulo-03-avanzado/` → Desarrollo de algoritmos, modelos [o la especialidad de tu curso].
* 📁 `proyectos-micro/` → Pequeños retos prácticos e integraciones de código individuales.

## 🚀 Sobre el Programa (SIC - UPV)
El **Samsung Innovation Campus** es un programa de capacitación tecnológica de alto rendimiento enfocado en la empleabilidad. A través de este curso en la UPV, no solo adquiero habilidades técnicas duras (*hard skills*), sino también metodologías de trabajo en equipo, resolución de problemas complejos y metodologías ágiles.
